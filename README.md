Un proiect de editor foto interactiv realizat in browser, folosind exclusiv HTML5 Canvas si JavaScript pur pentru manipularea pixelilor.

## Functionalitati principale
- Incarcare si descarcare a imaginilor prelucrate (.png).
- Selectie interactiva pe imagine si mutarea acesteia (folosind tasta Shift).
- Decupare (crop) si stergere (clear) a zonei selectate.
- Filtru alb-negru (grayscale) aplicabil direct pe selectie.
- Redimensionare a imaginii pastrand proportiile initiale.
- Adaugare de text personalizat peste imagine (culoare, dimensiune, coordonate X/Y).
- Generare histograma RGB in timp real pentru toata imaginea sau doar pentru zona selectata.

## Cum se ruleaza
Proiectul ruleaza exclusiv in browser. Descarcati fisierele si deschide `index.html` in orice browser (Chrome, Firefox, etc.). Nu necesita configurarea unui server.
